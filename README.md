# 🚗 Electric Vehicle Tableau Dashboard

## 📌 Project Objective
- Analyze Electric Vehicle dataset
- Understand EV adoption trends
- Identify top EV manufacturers
- Analyze EV distribution by state and year
- Create an interactive dashboard using Tableau
- Help decision makers understand EV market growth

---

## 📊 Dataset Overview
The dataset contains information about electric vehicles including:
- Model Year
- Vehicle Make & Model
- Electric Vehicle Type (BEV / PHEV)
- Electric Range
- State and Location
- CAFV Eligibility
- Base MSRP  
**Total Records:** 150,446 vehicles

---

## 🧹 Data Cleaning & Preparation
Steps performed before dashboard creation:
- Removed duplicate records
- Handled missing values
- Converted data types
- Standardized column names
- Filtered necessary columns
- Prepared dataset for Tableau visualization

---

## 🔑 Key KPIs
- **Total Electric Vehicles:** 150,446  
- **Average Electric Range:** 67.86 miles  
- **Total BEV Vehicles:** 116,771 (77.62%)  
- **Total PHEV Vehicles:** 33,675 (22.38%)  

---

## 📈 Dashboard Visualizations
Charts used in Tableau dashboard:
- **Line Chart** – Total Vehicles by Model Year
- **Map Visualization** – Total Vehicles by State
- **Bar Chart** – Top 10 Vehicle Manufacturers
- **Pie Chart** – CAFV Eligibility Distribution
- **Table View** – Total Vehicles by Model
- **Interactive Filters** – EV Type, State, Model

---

## 🚘 Insights
- Tesla dominates with **68,960 vehicles (52.71%)**
- Top models:  
  - Model Y – 28,501  
  - Model 3 – 27,708  
  - Nissan Leaf – 13,187  
- CAFV Eligibility:  
  - Eligible – 42.76%  
  - Not Eligible – 10.92%  
  - Unknown – 46.33%  
- EV adoption shows strong growth post‑2015
